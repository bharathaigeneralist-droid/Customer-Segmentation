# Customer-Segmentation
Developed a customer segmentation model using K-Means clustering to group customers based on spending behavior and income patterns. Applied data preprocessing, feature scaling, and model evaluation techniques to ensure accurate clustering. Generated actionable business insights to support targeted marketing and improve customer engagement.

## Dataset Used
<a href="https://github.com/bharathaigeneralist-droid/Customer-Segmentation/blob/main/Mall_Customers.csv">Data</a>

## Key Features
- End-to-end data pipeline: data loading, cleaning, preprocessing, and transformation
- Feature engineering with categorical encoding (Gender → Numeric)
- Scalable feature standardization using StandardScaler
- Optimal cluster detection using Elbow Method (WCSS)
- Model evaluation using Silhouette Score for clustering quality
- Interactive visualizations using Matplotlib & Seaborn
- Dimensionality reduction using PCA for advanced visualization
- Business-driven cluster labeling (Premium, Budget, Impulse Buyers, etc.)

- ## Benefits 
- Enables targeted marketing strategies based on customer segments
- Improves customer retention through personalized campaigns
- Identifies high-value customers for revenue optimization
- Helps businesses reduce marketing costs by focusing on the right audience
- Supports data-driven decision making instead of intuition-based strategies

- ## Novelty
- Combines unsupervised learning with business interpretation, not just clustering
- Uses Silhouette Score + Elbow Method for robust cluster validation
- Converts raw clustering output into actionable business personas
- Applies PCA visualization to simplify high-dimensional insights
- Bridges gap between technical ML model and real-world business application
